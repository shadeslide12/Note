# zjui-post Windows 部署指南

## 方法一：自动部署（推荐）

### 1. 运行部署脚本

在项目根目录打开 PowerShell，执行：

```powershell
.\deploy_windows.ps1
```

脚本会自动：
- 创建 `zjui-post-deploy` 目录
- 复制可执行文件和项目 DLL
- 使用 `windeployqt` 收集 Qt 依赖
- 复制 VTK、HDF5 等第三方库
- 生成 README.txt

### 2. 打包分发

```powershell
Compress-Archive -Path zjui-post-deploy -DestinationPath zjui-post-v1.0-win64.zip
```

将生成的 `.zip` 文件发给用户即可。

## 方法二：手动部署

### 必需文件清单

#### 1. 程序文件
```
cfd_post.exe
post_base_lib.dll
ColorMaps.json
```

#### 2. Qt 库（Release 版本）
```
Qt5Core.dll
Qt5Gui.dll
Qt5Widgets.dll
platforms/qwindows.dll  # 必须放在 platforms 子目录
```

#### 3. VTK 库
```
vtkCommonCore-9.1.dll
vtkCommonDataModel-9.1.dll
vtkCommonExecutionModel-9.1.dll
vtkGUISupportQt-9.1.dll
vtkRenderingCore-9.1.dll
vtkRenderingOpenGL2-9.1.dll
vtkRenderingFreeType-9.1.dll
vtkInteractionStyle-9.1.dll
... (其他 VTK DLL，见部署脚本)
```

#### 4. HDF5 库
```
hdf5.dll
hdf5_hl.dll
zlib.dll
szip.dll (可选)
```

#### 5. MSVC 运行时
```
vcruntime140.dll
msvcp140.dll
vcruntime140_1.dll
```

### 目录结构
```
zjui-post-deploy/
├── cfd_post.exe
├── post_base_lib.dll
├── ColorMaps.json
├── Qt5Core.dll
├── Qt5Gui.dll
├── Qt5Widgets.dll
├── vtkCommonCore-9.1.dll
├── ... (其他 DLL)
├── platforms/
│   └── qwindows.dll
└── README.txt
```

## 用户端安装说明

### 系统要求
- Windows 10/11 (64-bit)
- 无需安装 Qt、VTK 等开发环境

### 运行步骤
1. 解压 `zjui-post-v1.0-win64.zip`
2. 双击 `cfd_post.exe` 启动程序
3. 如果提示缺少 MSVC 运行时，下载安装：
   https://aka.ms/vs/17/release/vc_redist.x64.exe

## 故障排查

### 问题 1: 提示缺少 DLL
**解决方案**: 使用 [Dependency Walker](https://www.dependencywalker.com/) 检查缺少哪个 DLL，然后从编译环境复制到程序目录。

### 问题 2: 程序启动后立即崩溃
**可能原因**:
- Qt 平台插件缺失 → 确保 `platforms/qwindows.dll` 存在
- DLL 版本不匹配 → 确保所有 DLL 都是 Release 版本（不是 Debug 版本）

### 问题 3: 图标不显示
**解决方案**: 确保 `ColorMaps.json` 和程序在同一目录。

## 高级：创建安装程序

可以使用 [Inno Setup](https://jrsoftware.org/isinfo.php) 或 [NSIS](https://nsis.sourceforge.io/) 创建专业的安装程序。

示例 Inno Setup 脚本：

```ini
[Setup]
AppName=zjui-post
AppVersion=1.0
DefaultDirName={pf}\zjui-post
DefaultGroupName=zjui-post
OutputDir=.
OutputBaseFilename=zjui-post-setup

[Files]
Source: "zjui-post-deploy\*"; DestDir: "{app}"; Flags: recursesubdirs

[Icons]
Name: "{group}\zjui-post"; Filename: "{app}\cfd_post.exe"
Name: "{commondesktop}\zjui-post"; Filename: "{app}\cfd_post.exe"
```

---

## 注意事项

1. **不要混用 Debug 和 Release 版本的 DLL**
2. **确保 Qt 平台插件在 `platforms/` 子目录中**
3. **VTK DLL 版本必须与编译时一致（9.1.0）**
4. **如果目标机器没有 MSVC 运行时，需要包含或提示用户安装**
